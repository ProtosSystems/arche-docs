# Rate Limits

Arche enforces request limits to maintain system stability.

## Headers

```http
X-RateLimit-Limit
X-RateLimit-Remaining
X-RateLimit-Reset
Retry-After
```

When the rate limit is exceeded the API returns HTTP `429`.
