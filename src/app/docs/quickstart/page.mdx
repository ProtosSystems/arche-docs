# Quickstart

This guide shows how to retrieve financial statements from Arche in under a minute.

## Step 1 - Generate an API Key

Create an API key from the Arche dashboard.

All requests require an authorization header:

`Authorization: Bearer YOUR_API_KEY`

## Step 2 - Retrieve a Company

Example request:

`GET https://api.arche.fi/v1/companies?ticker=AAPL`

Example response:

`{"data":{"company_id":"6f6c9e1d","ticker":"AAPL","name":"Apple Inc."}}`

Save the `company_id`.

## Step 3 - Retrieve Financial Statements

Example request:

`GET /v1/companies/{company_id}/statements?statement_type=income`

Example response:

`{"page":1,"page_size":50,"total":1,"items":[{"statement_id":"stmt_123","statement_date":"2023-09-30","revenue":"383285000000","net_income":"96995000000"}]}`

Values are returned as **decimal strings** to preserve precision.

## Step 4 - Query as of Time

Arche allows queries using an `as_of` parameter.

Example:

`GET /v1/companies/{company_id}/statements?as_of=2024-01-01`

This returns the latest statement versions available on that date.
